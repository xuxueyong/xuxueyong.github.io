Swift 入门 第 7 课 集合-字典

字典是一个用来存储多个相同数据类型的集合。通过唯一的键 key， 找到对应的 value 值。 字典是无序的数据集合。和我们使用实物字典的时候很相似。

注意：
Swift 的 Dictionary 类型 可以桥接到 Objective-C 中的 NSDictionary 类型。

字典的语法

Dictionary<Key, Value>, Key 代表键的类型， Value 代表值得类型。

注意：
字典的 Key 类型必须遵守 Hasable 协议， 像 Set 的值类型一样。

简化写法 [Key : Value]
