Swift 入门 第 5 课 集合-数组

Swift 提供了 Array Dictionary Set 3 种集合类型。 Array 是有序集合， Dictionary 是键值对集合，Set 是无需集合。

Swift 集合中存储的数据类型必须明确， 这意味着我们存储数据和取数据都不会出类型错误。

三种集合类型， 都被实现为泛型集合， 后面会讲到泛型。

集合可变性

三种类型被声明为 var, 可以往集合中添加数据， 和从集合中删除和更新数据。 若被声明为 let 则不能改变集合的大小和内容。

注意
我们不需要集合的内容发生改变时， 就可以声明为 let，Swift 编译器优化创建的集合。

数组

可以存储相同类型的数据， 相同的数据可以存储多份。
